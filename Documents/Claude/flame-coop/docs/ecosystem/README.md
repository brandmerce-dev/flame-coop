# Eduweby Websites

## One-line description

A custom Next.js + Sanity CMS website platform for Christian homeschool cooperatives and microschools — built with ADA Tier 1 compliance standard, connected to Eduweby Enrollment OS via an embedded inquiry form.

## Current state

Live and in production. The Flame Christian Cooperative, Inc. (Ponte Vedra Beach, FL) is the first and currently only live client site built on this platform. The site is fully operational with Sanity CMS content management, Eduweby Enrollment OS form integration, and WCAG 2.1 Level AA accessibility compliance.

The codebase is designed as a repeatable template: each new co-op client deployment is expected to fork or redeploy from this base, configured with client-specific Sanity content, branding tokens, and Eduweby tenant credentials.

## The one-paragraph pitch

Eduweby Websites is the public-facing website layer of the Eduweby suite — a custom-built Next.js platform delivering conversion-focused, WCAG 2.1 Level AA accessible marketing sites for Christian homeschool cooperatives and microschools. Every site ships with ADA Tier 1 compliance built in (semantic HTML, keyboard navigation, color contrast, accessibility widget, skip link, and a live Accessibility Statement), with ADA Tier 2 legal documentation available as a $1,600 add-on. Sites are powered by Sanity CMS for non-technical content management, deployed to Vercel for global edge performance, and connect directly to Eduweby Enrollment OS via an embedded inquiry form — turning every "Request Info" click into a tracked enrollment lead in the Eduweby backend. The platform is built from a design system and content intake process ("website minutes") that allows rapid deployment for each new co-op client.
